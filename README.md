# bioinformatics-workflow-android

Capstone project for MS in Computer Science and Systems

Recent advances in nanopore sequencing have revolutionized genomics by improving accuracy, portability, and affordability. Allowing for cancer diagnostics in resource intensive environments. While nanopore devices have become portable over the years, they still rely on traditional computing infrastructure. This project addresses this limitation by enabling nanopore gene sequencing and analysis pipelines on Android-based mobile devices, such as smartphones and tablets.

The core of this implementation involves the creation of a localized Linux environment on the Android Operating System using Termux. By configuring this environment to support the Biodept Workflow Builder we are able to run cancer diagnostics on portable consumer grade Android devices. This methodology allows for sophisticated genomic analysis in remote locations without requiring extensive laboratory or clinical infrastructure. Our results demonstrate that mobile devices can serve as viable computational hubs for real-time sequencing, significantly lowering the barrier to entry for decentralized molecular diagnostics and field-based clinical research.

Keywords: Basecaller, Nanopore Sequencing, Android Open Source Project(AOSP) , Linux
